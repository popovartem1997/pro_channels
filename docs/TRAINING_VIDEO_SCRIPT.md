# Обучающее видео ProChannels (сценарий)

## Цель видео
Показать путь “с нуля до результата” за 10–15 минут:
подключили канал → включили рекламу → получили заявку → оплатили → автопубликация → акт → статистика.

## 0. Вступление (0:00–0:30)
- Что такое ProChannels
- Для кого: владельцы пабликов и команды, рекламодатели
- Что покажем: публикации, предложка, парсинг, реклама, документы, ОРД

## 1. Регистрация и кабинет (0:30–1:30)
- Регистрация / вход
- Обзор дашборда: каналы, посты, боты, парсинг, реклама, счета
- Где “Быстрый старт”

## 2. Подключение канала (1:30–4:00)
- Каналы → Добавить
- Ввод токена/ID
- “Проверить подключение”
- Где редактируются подписи (TG/MAX/VK)

## 3. Витрина рекламы: включить и поставить цену (4:00–5:30)
- Открыть канал → Редактировать
- “Принимать рекламу” = ON
- Цена за размещение (₽)
- Проверка: Каталог каналов

## 4. Публикация постов и планировщик (5:30–7:30)
- Посты → Создать
- Выбор каналов, дата, повтор
- Публикация сейчас/планирование
- Где смотреть результат: детальная страница поста + PublishResult

## 5. Предложка-бот (7:30–9:00)
- Боты предложки → Создать
- Куда приходят заявки на модерацию
- Как выглядит статус обработки для пользователя

## 6. Парсинг + AI (9:00–11:00)
- Парсинг → источники + ключевики
- Список найденного контента
- AI‑рерайт: как запустить и где смотреть результат

## 7. Реклама: заказ → счет → оплата → автопост → акт (11:00–14:30)
- Рекламодатель: регистрация
- Создать заказ: период + повтор (интервал дней)
- Выбор каналов + живой расчёт бюджета
- Владелец: “Одобрить” → создаётся счёт
- Оплата (TBank) → заказ становится active → автопост создаётся/планируется
- Завершение кампании → акт + PDF

## 8. ОРД (14:30–15:30)
- Где смотреть список ОРД регистраций
- Что автоматически подставляется в VK

## Финал (15:30–16:00)
- Куда писать за поддержкой
- Что планируется дальше (Instagram, расширенная аналитика, автопрайс и т.д.)
