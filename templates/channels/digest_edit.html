Как это работает
  • Погода — Open-Meteo (без ключа), по широте/долготе.
  • Солнце — публичный API sunrise-sunset.org.
  • Праздники — библиотека holidays по коду страны.
  • Цитата, слово, гороскоп — при включённых опциях и ключе DeepSeek в «Ключи API» текст генерируется моделью; иначе подставляются нейтральные шаблоны.
  • Картинка — сервис picsum.photos по дате и seed (DeepSeek не генерирует изображения; для своих картинок позже можно подключить отдельный API).

Celery Beat должен выполнять задачу channels.tasks.channel_morning_digest_tick (раз в ~2 мин.) — после деплоя выполните python manage.py setup_periodic_tasks.

Если токен канала ещё не настроен, по расписанию всё равно создаётся черновик; в очередь на публикацию пост попадёт только при настроенном токене.
